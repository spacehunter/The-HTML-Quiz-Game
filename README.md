# HTML5 Elements Quiz

Test your HTML5 knowledge! See how many HTML5 elements you can name in 5 minutes.

## About

This quiz challenges you to name as many HTML5 elements as you can within a 5-minute time limit. It's a fun way to test and improve your knowledge of HTML5 elements.

Originally created by [Kevin Sweeney](https://kevinsweeney.info) and converted to AngularJS by Joe Hillman, this version has been completely modernized with vanilla JavaScript and ES6+ features.

## How to Play

1. Click "Let's do this!" to start the quiz
2. Type HTML element names (like `div`, `span`, `article`, etc.)
3. Correct answers are automatically logged
4. You have 5 minutes to name as many as you can
5. See which elements you missed at the end

## Running Locally

### Simple Method (No Build Required)
Just open `index.html` in your web browser!

### Using a Local Server
```bash
# Install dependencies (optional)
npm install

# Start a local server
npm start
```

Then navigate to `http://localhost:3000` in your browser.
